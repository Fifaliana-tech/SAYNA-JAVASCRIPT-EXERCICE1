JAVASCRIPT Exo-1

Variables et types

Introduction :

Cet exercice est spécialement conçu pour les débutants et vous apprendra les bases de la programmation en JavaScript. En même temps que vous avancez dans le cours, vous devez vous exercer pour comprendre les bases de JavaScript.


Objectif Pédagogique:

Cela vous permettra de pratiquer et d’assimiler les bases de ce langage de programmation. Vous devez vous entraîner suffisamment avec les exercices pour être à l'aise avec les bases de JavaScript avant de passer aux projets.
